The file manager shows the current workspace tree. The footer switches workspaces, sorting, and expansion; the top controls create and import content.

## Basic operations

- New files and folders receive a non-conflicting name.
- Select a file to open it in a tab; select a folder to expand or collapse it.
- Context menus rename, copy, cut, paste, delete, reveal in the system file manager, and run remote sync operations.
- Drag files and folders into a destination, or drag supported content into the editor.
- Platform keyboard shortcuts work for copy, cut, paste, delete, and rename.

## Multiple selection

Use the platform's multi-select controls, then copy, cut, or delete the selection. Batch deletion asks for confirmation and may also delete remote copies when sync is active.

## Import Markdown

Select a directory to recursively copy Markdown and related images into the workspace. Check for name collisions first. NoteGen reports the number imported when finished.

## Workspaces

- The default workspace is inside NoteGen's application-data directory.
- A custom workspace can be any local directory with read/write permission.
- History keeps recently used paths for quick switching and can be cleared.
- Restart after changing a workspace. Switching does not move files from the previous workspace.

## Sort and expand

Sort by name, creation time, or modification time in ascending or descending order. Expand or collapse all folders at once.

## Knowledge base

File and folder menus can:

- calculate or delete vectors manually;
- toggle automatic vectors for one file;
- exclude a file from the knowledge base;
- batch-calculate or remove folder vectors;
- display folder index statistics.

The Skills directory is loaded as workspace Skills rather than ordinary knowledge-base content.

## Sync

The primary provider can be GitHub, Gitee, GitLab, Gitea, S3, or WebDAV. Items show relevant status, and upload, pull, and delete use the selected provider. Git providers can open the remote repository; S3 and WebDAV organize files under object/path prefixes.

## Export and unsupported files

Export Markdown, HTML, plain text, JSON, or PDF from the editor or file actions. Files that NoteGen cannot edit can still be opened in a system application.

## Mobile

Mobile uses touch menus instead of right-click and simplifies some batch and filesystem operations. Workspaces and Markdown remain compatible with desktop, subject to mobile OS directory permissions.
