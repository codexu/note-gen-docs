## I. Chat Conversation Feature

Chat is the core interface for interacting with the AI assistant, supporting Q&A, discussions, and getting advice.

### 1.1 Accessing Chat

- **Desktop**: Click the Chat icon in the left sidebar
- **Mobile**: Switch to the Chat tab

### 1.2 Chat and Agent Modes

On the left side of the bottom toolbar, you can switch between **Chat** and **Agent** modes:

#### Chat Mode (üí¨ Conversation)

Chat mode is the standard conversation mode, suitable for daily communication and simple Q&A.

**Features**:
- **One Question, One Answer**: Send a question and AI responds directly
- **Quick Response**: No complex reasoning process, results are output immediately

**Suitable for**:
- Asking knowledge-based questions
- Getting suggestions and opinions
- Text polishing and editing
- Simple translation tasks

#### Agent Mode (ü§ñ Intelligent Agent)

Agent mode uses the ReAct framework, allowing AI to autonomously call tools to complete complex tasks.

**Features**:
- **Think-Act-Observe**: AI displays the complete thought process and execution steps
- **Tool Calling**: Can call tools like search, create notes, modify files, etc.
- **Multi-step Iteration**: Up to 15 iterations to complete complex tasks
- **User Confirmation**: Certain important operations will request your confirmation

**Execution Process Display**:
```
Thought: [AI's thought process]
Action: [Tool name being called]
Action Input: [Tool parameters]
Observation: [Tool execution result]
```

**Suitable for**:
- Searching and organizing notes
- Creating or modifying files
- Complex tasks requiring multiple steps
- Tasks requiring MCP services

## II. Note Recording Feature

The Record feature is used to quickly record inspiration, fragments, and other content. These notes automatically become context for AI conversations.

### 2.1 Recording Notes

1. Click the **note icon** in the left sidebar to enter the recording interface
2. Click the **text record button** (copy icon)
3. In the dialog that appears:
   - Directly type or paste text content
   - Check "Auto-read clipboard" to automatically get clipboard content
   - Click **Save** to complete recording

### 2.2 Managing Notes

- **View Notes**: View all saved content in the notes list
- **Multi-select Operations**: Enter multi-select mode to batch delete notes
- **Tag Categories**: Add tags to notes for better organization

### 2.3 Notes and AI Integration

Your saved notes are used by AI in the following scenarios:

- During Chat conversations, notes serve as context reference
- When using smart placeholders, note content is automatically included
- In the editor, you can quickly insert notes into articles

## III. Editor AI Tools

The article editor provides multiple AI-assisted writing tools.

### 3.1 Continue Writing

Let AI continue generating text based on the content at the current cursor position.

**How to Use**:
1. Place the cursor where you want to continue writing
2. Click the **Continue Writing button** in the toolbar
3. AI will stream the generated content
4. Click the stop button if you need to interrupt

### 3.2 Translation

Quickly translate selected text.

**How to Use**:
1. Select the text you want to translate
2. Click the **Translation button** in the toolbar
3. The translation result will be automatically inserted into the editor

### 3.3 Insert Notes

Quickly insert saved notes at the current editing position.

**How to Use**:
1. Place the cursor where you want to insert
2. Click the **Insert Notes button** in the toolbar
3. Select the note you want to insert from the list
4. The note content will be inserted into the article, and the original note will be deleted

## VI. Common Scenario Examples

### Scenario 1: Let AI Help You Rewrite a Paragraph

1. Select the text you want to rewrite in the editor
2. Open Chat, ensure the current file is linked
3. Enter: "Help me rewrite this text to make it more concise"
4. AI will provide rewrite suggestions based on the selected content

### Scenario 2: Generate Article Outline Based on Notes

1. First record some key points and materials in the notes feature
2. Open Chat, notes will automatically serve as context
3. Enter: "Generate an article outline based on my notes"
4. AI will analyze the note content and generate an outline

### Scenario 3: Continue Creating When Facing Writer's Block

1. Place the cursor where you need to continue
2. Click the **Continue Writing** button in the toolbar
3. AI will continue writing based on context
4. You can continue editing or ask AI to generate again
