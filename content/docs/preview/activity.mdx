活动统计用于回顾自己在 NoteGen 中的使用节奏。它记录活动次数，不会因为原笔记或记录后来被删除而抹去历史事件。

<img src="https://files.seeusercontent.com/2026/07/16/D2pz/desktop-activity.jpeg" alt="桌面端活动统计" className="rounded-xl border" />

## 查看活动

- 桌面端可从活动面板查看热力图和周累计数据。
- 移动端可从“我的”页面打开活动抽屉。
- 点击热力图中的日期，可以查看当天的活动时间线。

## 活动类型

| 类型 | 示例 |
| --- | --- |
| 记录 | 新建或保存文本、图片、链接、录音等记录 |
| 写作 | 创建或编辑 Markdown 文件 |
| 对话 | 与 Chat 或 Agent 交互 |

热力图显示的是活动次数，不等同于字数、工作时长或完成任务数。统计保存在本地；启用记录与配置同步后，相关数据可能随同步数据一同传输。
