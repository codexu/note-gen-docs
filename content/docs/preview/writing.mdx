NoteGen 使用 Tiptap 3 构建单一的所见即所得编辑体验：编辑器直接渲染 Markdown 结构，不需要在源码、预览和分屏模式之间切换。

## 打开与管理文件

- 从文件管理器新建或打开 Markdown 文件。
- 多个文件以标签页形式打开；记录详情也可以作为标签页查看。
- 内容自动保存到当前工作区。文件同步是否自动执行取决于同步设置。
- 常见文本代码文件可以在编辑器中打开；不支持的格式会显示不可编辑提示。

## 编辑能力

编辑器支持标题、段落、加粗、斜体、删除线、下划线、高亮、引用、链接、无序/有序/任务列表、行内代码、代码块、表格、图片和水平线。

- 输入 Markdown 标记可触发对应格式。
- 输入 `/` 打开斜杠命令菜单。
- 使用块左侧拖拽手柄调整内容顺序。
- 表格拥有行列添加、删除和对齐等浮动操作。
- 图片支持粘贴、拖入、选择上传、缩放和路径转换。

## 数学公式与图表

- 行内公式使用 `$...$`，块公式使用 `$$...$$`，由 KaTeX 渲染。
- Mermaid 代码块可渲染流程图、时序图、类图、状态图、ER 图、甘特图、饼图和旅程图。
- 点击公式或 Mermaid 内容可打开专用编辑界面。

## AI 写作

### 内联补全

开启底部的 AI 补全后，编辑器会根据光标前后的内容生成建议。使用“设置 → 快捷键”中配置的按键接受或取消建议。

### 选区工具

选中文字后，浮动工具栏提供润色、精简、扩写和翻译。生成内容会以流式方式显示，可接受或放弃。

### Agent 修改

Agent 可以读取当前文件并提出修改。变更会显示为 Diff，确认后才写入编辑器；涉及多个文件时可在变更面板逐项检查。

## 引用记录与文件

- 将记录拖入编辑器可插入文字、图片或文件引用。
- 将文件管理器中的文本文件拖入编辑器可创建链接或插入内容，具体行为取决于文件类型。
- 在对话中引用选中文字，可以让 Chat 或 Agent围绕该段内容工作。

## 搜索、大纲与快捷键

- 查找/替换支持上一个、下一个和大小写匹配。
- 大纲根据标题生成，可放在编辑器左侧或右侧。
- 所有编辑器快捷键都可在“设置 → 快捷键”中查看和修改，并会检测与全局快捷键的冲突。

## 导出

编辑器导出菜单支持 Markdown、HTML、纯文本、JSON 和 PDF。文件管理器右键菜单也可提供 Markdown 文件导出操作。PDF 由当前渲染内容生成，复杂分页、远程图片和超宽表格建议导出后检查。

## 移动端

移动端使用相同 Markdown 数据，提供适合触控的选区上下文栏、写作工具栏和“更多”菜单。桌面端的部分悬浮操作会在移动端改为底部抽屉。
