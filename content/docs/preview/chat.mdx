## 一、Chat 对话功能

Chat 是你与 AI 助手交互的核心界面，可以进行问答、讨论、获取建议等。

### 1.1 进入 Chat

- **电脑端**：点击左侧边栏的 Chat 图标
- **移动端**：切换到 Chat 标签页

### 1.2 Chat 与 Agent 模式

在底部工具栏左侧，你可以切换 **Chat** 和 **Agent** 两种对话模式：

#### Chat 模式（💬 对话）

Chat 模式是标准的对话模式，适合日常交流和简单问答。

**特点**：
- **一问一答**：发送问题，AI 直接给出回答
- **快速响应**：没有复杂的推理过程，直接输出结果

**适合场景**：
- 询问知识性问题
- 获取建议和意见
- 文字润色和修改
- 简单的翻译任务

#### Agent 模式（🤖 智能代理）

Agent 模式使用 ReAct 框架，AI 可以自主调用工具来完成复杂任务。

**特点**：
- **思考-行动-观察**：AI 会展示完整的思考过程和执行步骤
- **工具调用**：可以调用搜索、创建笔记、修改文件等工具
- **多步迭代**：最多执行 15 次迭代来完成复杂任务
- **用户确认**：某些重要操作会请求你的确认

**执行过程显示**：
```
Thought: [AI 的思考过程]
Action: [调用的工具名称]
Action Input: [工具参数]
Observation: [工具执行结果]
```

**适合场景**：
- 搜索和整理笔记
- 创建或修改文件
- 需要多步操作的复杂任务
- 需要调用 MCP 服务的任务

### 1.3 Chat 界面介绍

Chat 界面主要由以下几个部分组成：

#### 顶部工具栏

顶部工具栏分为左右两个区域，提供了一系列 AI 对话增强功能：

**左侧工具栏**

| 工具 | 图标 | 功能说明 |
|------|------|----------|
| **关联笔记** | 🔗 | 开启后，当前标签下的笔记会作为 AI 对话的上下文；按钮会显示已关联的笔记数量 |
| **关联文件** | @ | 点击后可选择要关联的文件或文件夹，AI 将基于这些文件内容进行对话 |
| **MCP 服务器** | 🖥️ | 选择要启用的 MCP（Model Context Protocol）服务器，扩展 AI 的能力范围 |
| **剪贴板监听** | 📋 | 开启后，AI 会自动监听剪贴板内容变化（默认开启） |

**右侧工具栏**

| 工具 | 图标 | 功能说明 |
|------|------|----------|
| **清除上下文** | 🗑️ | 插入一条系统消息，之后的对话将不再携带此消息之前的历史内容 |
| **清空对话** | 🧹 | 清空当前标签下的所有对话历史 |

#### 输入区域（底部）

输入框位于界面底部，提供以下功能：

- **输入框**：输入你要发送给 AI 的内容
- **发送消息**：点击发送按钮或按 `Enter` 键
- **换行**：按 `Shift + Enter` 换行
- **浏览历史**：按 `↑` / `↓` 方向键浏览历史输入

#### 底部工具栏

| 工具 | 功能说明 |
|------|----------|
| **模型选择** | 选择使用哪个 AI 模型进行对话 |
| **提示词模板** | 使用预设的提示词模板，快速设置 AI 的角色或任务 |
| **回复语言** | 设置 AI 回复使用的语言 |
| **上传图片** | 上传图片让 AI 识别或分析 |

### 1.4 AI 智能输入建议

在输入框中，你可以使用以下快捷功能：

- 按 `Tab` 键：插入智能占位符（占位符会根据你的笔记、关联文件等内容自动生成）
- 按 `↑` / `↓` 方向键：浏览历史输入

## 二、笔记记录功能

笔记（Record）功能用于快速记录灵感、片段等内容，这些笔记会自动成为 AI 对话的上下文。

### 2.1 记录笔记

1. 点击左侧边栏的**笔记图标**进入记录界面
2. 点击**文本记录按钮**（复制图标）
3. 在弹出的对话框中：
   - 直接输入或粘贴文本内容
   - 勾选「自动读取剪贴板」可自动获取剪贴板内容
   - 点击**保存**完成记录

### 2.2 管理笔记

- **查看笔记**：在笔记列表中查看所有已保存的内容
- **多选操作**：进入多选模式可批量删除笔记
- **标签分类**：为笔记添加标签便于整理

### 2.3 笔记与 AI 的配合

你保存的笔记会在以下场景中被 AI 使用：

- Chat 对话时，笔记会作为上下文参考
- 使用智能占位符时，笔记内容会被自动包含
- 在编辑器中可以快速插入笔记到文章中

## 三、编辑器 AI 工具

在文章编辑器中，提供了多个 AI 辅助写作工具。

### 3.1 继续写作

让 AI 基于当前光标位置的内容，继续生成后续文本。

**使用方法**：
1. 将光标放在想要继续写作的位置
2. 点击工具栏的**继续写作按钮**
3. AI 会流式输出生成的内容
4. 如需中断，点击停止按钮

### 3.2 翻译

快速翻译选中的文本。

**使用方法**：
1. 选中需要翻译的文本
2. 点击工具栏的**翻译按钮**
3. 翻译结果会自动插入到编辑器

### 3.3 插入笔记

快速将保存的笔记插入到当前编辑位置。

**使用方法**：
1. 将光标放在想要插入的位置
2. 点击工具栏的**插入笔记按钮**
3. 在弹出的列表中选择要插入的笔记
4. 笔记内容会插入到文章中，原笔记会被删除

## 四、提示词模板

提示词模板可以帮你快速设定 AI 的角色或任务类型。

### 4.1 使用提示词模板

1. 在 Chat 输入框上方，点击**提示词模板按钮**（🎭 图标）
2. 从下拉列表中选择一个模板
3. 该模板会应用到后续的 AI 对话中

### 4.2 常见模板类型

- **写作助手**：帮助优化文章、提供写作建议
- **翻译助手**：专注于翻译任务
- **分析助手**：分析和总结内容
- **自定义模板**：你可以创建自己的提示词模板

## 五、使用技巧

### 5.1 提升对话质量

- **使用关联文件**：让 AI 基于当前文件内容回答问题
- **善用笔记**：将相关背景记录为笔记，AI 会自动参考
- **设置回复语言**：确保 AI 用你期望的语言回复

### 5.2 快捷操作

| 按键 | 功能 |
|------|------|
| `Enter` | 发送消息 |
| `Shift + Enter` | 换行 |
| `Tab` | 插入智能占位符 |
| `↑` / `↓` | 浏览输入历史 |

### 5.3 多模态输入

- **文字**：直接输入问题或内容
- **图片**：上传图片让 AI 识别、分析或描述
- **文件**：关联文件让 AI 基于文件内容对话

## 六、常见场景示例

### 场景 1：让 AI 帮你改写一段文字

1. 在编辑器中选中要改写的文字
2. 打开 Chat，确保关联了当前文件
3. 输入：「帮我改写这段文字，使其更加简洁」
4. AI 会基于选中内容给出改写建议

### 场景 2：基于笔记生成文章大纲

1. 先在笔记功能中记录一些要点和素材
2. 打开 Chat，笔记会自动作为上下文
3. 输入：「基于我的笔记，生成一篇文章大纲」
4. AI 会分析笔记内容并生成大纲

### 场景 3：写作遇到瓶颈时继续创作

1. 将光标放在需要继续的位置
2. 点击工具栏的**继续写作**按钮
3. AI 会根据上下文续写内容
4. 你可以继续编辑或让 AI 再次生成
