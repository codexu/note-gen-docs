Record captures material before it becomes a document. Organize turns selected records into structured Markdown. On desktop, open Record from the sidebar; mobile has a dedicated Record screen.

## Seven record types

| Type | Content and usage |
| --- | --- |
| Text | Type, paste, or capture clipboard text |
| Voice | Record live or import audio and transcribe it with STT |
| Scan | Capture a screen area and extract content with OCR or VLM |
| Image | Store an image with recognized text or a description |
| Link | Save a URL, title, and description and reopen the source |
| File | Store an attachment path and description |
| Todo | Create, edit, and complete a task |

Change record-control order and visibility under Settings → Record. Voice, OCR, VLM, and descriptions require the corresponding models or system capability.

## Tags and destination

- Every record belongs to a tag; select the destination while creating it.
- Create, rename, reorder, and delete tags.
- Deleting a tag moves its records to Trash rather than removing them permanently.

## Browse, search, and filter

- Switch among list, compact, and card views.
- Search record text, descriptions, and links.
- Filter by all time, today, the last 7 days, or the last 30 days.
- Select one or more record types.
- Combine tag, search, time, and type filters.

## Record actions

Open details to edit content, move a tag, open the source, delete, or restore. Multi-select exposes the batch operations currently available in the interface. Deleted records enter Trash, where they can be restored, permanently deleted, or cleared together.

## AI organization

1. Open Organize and select a template.
2. Choose a tag, time range, and individual records, or use quick mode.
3. Choose one or multiple output files and a destination directory.
4. Optionally remove model thinking and review the selected records.
5. AI processes text, scans, images, links, files, voice records, and todos into Markdown.

Organization does not automatically delete the source records. Continue editing the generated files in Writing.

## Templates

A template includes its name, enabled state, default range, and prompt. Create separate templates for meeting minutes, study summaries, research, or daily reports. Specify output structure, source links, image placement, and todo formatting in the prompt.

## Chat and Agent

- Link a tag from the chat toolbar to include its records in context.
- Agent can query, create, update, and organize records through tools; write and delete operations follow approval rules.
- Drag a record into the editor to insert it into an existing note.

## Sync and privacy

Records are stored in a local database. Records and Settings sync uploads records, tags, chats, and permitted settings to the selected provider. Online transcription, VLM, and description models receive the relevant audio, image, or text.
