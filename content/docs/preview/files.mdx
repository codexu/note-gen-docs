## 界面布局

- **顶部工具栏**：向量数据库同步、远程仓库访问
- **操作工具栏**：新建文件、新建文件夹、批量导入
- **文件树区域**：显示当前工作区的文件和文件夹结构
- **底部状态栏**：工作区切换、排序方式、展开/折叠控制

## 基本操作

### 新建文件/文件夹

**方式一：工具栏按钮**
1. 点击顶部操作栏的「新建文件」或「新建文件夹」按钮
2. 输入文件名，按 Enter 确认

**方式二：右键菜单**
1. 在文件树空白处右键点击
2. 选择「新建文件」或「新建文件夹」
3. 输入文件名，按 Enter 确认

**方式三：文件夹内创建**
1. 右键点击目标文件夹
2. 选择「新建子文件」或「新建子文件夹」

> 提示：系统会自动生成唯一文件名，避免命名冲突

### 打开文件

- **单击**文件项：在编辑区打开文件
- **双击**文件夹：展开或折叠文件夹

### 重命名

**方式一：双击重命名**
1. 双击文件名进入编辑状态
2. 修改名称后按 Enter 确认，或按 Escape 取消

**方式二：右键菜单**
1. 右键点击文件，选择「重命名」
2. 修改名称后按 Enter 确认

> 提示：文件名中的空格会自动替换为下划线

### 删除文件

1. 右键点击文件，选择「删除」
2. 在确认对话框中点击确认

> 注意：删除云端同步的文件会同时删除远程仓库中的文件

### 移动文件

**方式一：剪贴板操作**
1. 右键点击文件，选择「剪切」
2. 右键点击目标文件夹，选择「粘贴」

**方式二：拖拽操作**
1. 按住文件项拖动
2. 拖拽到目标文件夹后释放

## 高级功能

### 批量导入

1. 点击操作栏的「导入」按钮
2. 在文件对话框中选择文件夹
3. 系统会自动递归导入所有 Markdown 文件和图片

导入完成后会显示成功导入的文件数量。

### 工作区管理

**切换工作区**
1. 点击底部状态栏的工作区名称
2. 从历史记录中选择或创建新工作区

**工作区类型**
- **默认工作区**：应用内置的默认存储位置
- **自定义工作区**：用户指定的任意文件夹

系统会保存最近使用的 10 个工作区历史。

### 排序方式

点击底部状态栏的排序按钮，可选择：

| 排序方式 | 说明 |
|---------|------|
| 按名称 | 文件名 A-Z 排序 |
| 按创建时间 | 文件创建时间排序 |
| 按修改时间 | 文件最后修改时间排序 |

支持升序/降序切换。

### 展开/折叠控制

- **单个文件夹**：点击文件夹图标展开/折叠
- **全部展开**：点击底部的「全部展开」按钮
- **全部折叠**：点击底部的「全部折叠」按钮

## 特殊功能

### 向量计算

文件管理器集成向量数据库功能，支持语义检索：

- **计算状态**：文件项上显示向量计算进度
- **索引统计**：文件夹显示已索引文件数量
- **手动触发**：可手动触发向量计算

### 云端同步

支持的同步平台：
- GitHub
- Gitee
- GitLab
- Gitea

云端文件会在文件项上显示云图标标识。

### Skills 文件夹

Skills 是一个特殊文件夹，用于管理应用技能文件：
- 带有特殊图标标识
- 不参与向量计算
- 支持创建和管理技能文件

## 移动端使用

移动端提供优化的交互体验：

- **操作菜单**：点击文件后的「...」按钮打开操作菜单
- **触摸优化**：更大的点击区域和手势支持
- **简化界面**：只显示常用操作，提升易用性
