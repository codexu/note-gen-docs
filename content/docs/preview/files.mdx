文件管理器显示当前工作区的目录树。底部可切换工作区、排序和展开状态，顶部提供新建与批量导入等操作。

## 基本操作

- 新建文件或文件夹时，NoteGen 会生成不冲突的名称。
- 单击文件在标签页中打开；单击文件夹展开或折叠。
- 右键菜单可重命名、复制、剪切、粘贴、删除、在系统文件管理器中查看和执行远程同步。
- 文件和文件夹可拖到目标目录；也可以拖入编辑器创建引用。
- 复制、剪切、粘贴、删除和重命名支持平台对应的键盘快捷键。

## 多选

使用系统多选方式选择多个文件或文件夹后，可批量复制、剪切或删除。批量删除会显示确认，且远程同步文件可能同时从当前同步平台删除。

## 批量导入

“导入 Markdown”选择一个目录后递归复制其中的 Markdown 和相关图片到当前工作区。导入前检查同名文件，完成后应用会显示成功数量。

## 工作区

- 默认工作区位于 NoteGen 应用数据目录。
- 自定义工作区可以是任意有读写权限的本地目录。
- 历史记录保存最近使用的路径，可快速切换或清空。
- 更改工作区后建议重启应用；切换不会自动迁移旧工作区文件。

## 排序与展开

文件树支持按名称、创建时间或修改时间排序，并可切换升序/降序。还可一键展开或折叠所有目录。

## 知识库

文件和文件夹菜单可以：

- 手动计算或删除向量；
- 为单个文件控制自动向量计算；
- 将文件排除在知识库之外；
- 对文件夹批量计算或删除向量；
- 查看文件夹索引统计。

Skills 目录不会作为普通知识库内容处理，而是用于加载工作区 Skills。

## 同步

当前主要平台可以是 GitHub、Gitee、GitLab、Gitea、S3 或 WebDAV。文件项会显示相关同步状态；上传、拉取和删除由当前平台处理。Git 平台可打开远程仓库，S3/WebDAV 使用对象或路径前缀组织文件。

## 导出与不支持的文件

Markdown 文件可从编辑器或文件菜单导出为 Markdown、HTML、纯文本、JSON、PDF 等格式。不支持编辑的文件仍可在系统应用中打开。

## 移动端

移动端使用触控菜单代替右键，并简化部分批量和文件系统操作。工作区、文件和 Markdown 内容与桌面端兼容，但受移动系统目录权限限制。
