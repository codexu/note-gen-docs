记录用于快速收集尚未成文的信息，整理用于把选中的记录生成结构化 Markdown。桌面端从左侧“记录”进入，移动端使用独立记录页面。

## 七种记录

| 类型 | 内容与用法 |
| --- | --- |
| 文本 | 手动输入、粘贴或从剪贴板快速记录 |
| 录音 | 实时录音或导入音频，通过 STT 转写 |
| 扫描 | 截图后使用本地 OCR 或 VLM 提取内容 |
| 图片 | 保存图片并生成识别文本或描述 |
| 链接 | 保存 URL、标题和说明，可打开原网页 |
| 文件 | 保存附件路径和描述，可从记录详情打开 |
| 待办 | 创建、编辑并勾选完成状态 |

记录入口的顺序和可见性可在“设置 → 记录设置”调整。录音、OCR、VLM 和描述生成需要对应模型或系统能力。

## 标签与保存位置

- 每条记录属于一个标签；创建时可选择目标标签。
- 标签支持创建、重命名、排序和删除。
- 删除标签时，其中的记录会进入回收站，而不是立即永久清除。

## 浏览、搜索与筛选

- 视图可切换为列表、紧凑或卡片。
- 可搜索记录正文、描述和链接。
- 时间筛选支持全部、今天、最近 7 天和最近 30 天。
- 类型筛选可同时选择一种或多种记录类型。
- 标签筛选与搜索、时间、类型条件可以组合使用。

## 记录操作

打开记录详情可编辑内容、移动标签、打开源文件/链接、删除或恢复。多选模式用于批量选择并执行当前界面提供的批量操作。删除的记录进入回收站，可单条或全部恢复，也可以永久删除或清空回收站。

## AI 整理

1. 打开整理面板并选择模板。
2. 选择标签、时间范围和具体记录；也可以使用快速模式。
3. 选择生成一个新文件或多个文件，以及目标目录。
4. 按需移除模型思考内容，并确认将使用的记录。
5. AI 按模板处理文本、扫描、图片、链接、文件、录音和待办，生成 Markdown。

整理只会读取本次选择的记录，不会自动删除原记录。生成后可以继续在编辑器中修改。

## 整理模板

模板包含名称、启用状态、默认范围和 Prompt。适合为会议纪要、学习总结、项目调研、日报等场景建立不同模板。模板 Prompt 应明确输出结构、是否保留来源链接、图片位置和待办格式。

## 与 Chat 和 Agent 配合

- 对话工具栏中的关联标签可以把当前标签记录加入上下文。
- Agent 可通过记录和标签工具查询、创建、更新或整理内容，执行写入和删除前会遵循权限确认。
- 将记录拖入编辑器，可快速插入到现有笔记。

## 同步与隐私

记录保存在本地数据库。启用“记录与配置同步”后，记录、标签、聊天和允许同步的设置会上传到当前同步平台。在线转写、VLM 或描述模型会收到相应音频、图片或文本。
