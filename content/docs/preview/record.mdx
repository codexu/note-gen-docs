## 记录与整理功能使用指南

NoteGen 的"记录"和"整理"功能旨在帮助你轻松收集碎片化信息，并利用 AI 将其整理成结构化的知识体系。

- **记录功能**：快速收集各类信息（文本、图片、链接、录音等），无需考虑组织和分类
- **整理功能**：AI 自动将散乱的记录整理成逻辑清晰、结构完整的文档

---

### 一、记录功能

#### 1.1 访问记录功能

**桌面端**：点击左侧边栏的"记录"标签

**移动端**：进入独立的"记录"页面

#### 1.2 支持的记录类型

NoteGen 支持以下 7 种记录类型：

##### 文本记录

快速记录文本内容，支持从剪贴板自动读取。

**创建方式**：
- 点击工具栏的文本按钮
- 复制文本后自动识别并创建
- 使用快捷键快速记录

**适用场景**：突然想到的点子、临时笔记、摘录段落等

##### 扫描记录（OCR）

通过截图自动识别图片中的文字内容。

**创建方式**：
- 点击工具栏的扫描按钮进行截图
- 系统自动进行 OCR 文字识别
- 支持保存截图和识别结果

**适用场景**：识别图片中的文字、扫描纸质文档、提取屏幕信息

##### 图片记录

保存图片并添加描述。

**创建方式**：
- 点击工具栏的图片按钮选择文件
- 支持常见的图片格式
- 可为图片添加描述文字

**适用场景**：保存设计参考、收集素材、记录可视化信息，在整理时会插入到笔记的合适位置

##### 链接记录

自动识别剪贴板中的链接并保存。

**创建方式**：
- 复制链接地址自动创建记录
- 自动获取网页标题
- 点击可直接跳转到原链接

**适用场景**：收藏网页、保存资源链接、待读文章列表

##### 文件记录

保存文件及其描述信息。

**创建方式**：
- 点击工具栏的文件按钮
- 选择任意类型的文件
- 添加文件说明或备注

**适用场景**：保存文档附件、归档资料文件

##### 录音记录

支持实时录音或上传音频文件，自动转写为文字。

**创建方式**：
- **单击**麦克风图标：开始实时录音
- **双击**麦克风图标：选择音频文件上传
- 系统自动进行语音转文字

**适用场景**：会议记录、语音备忘、讲座笔记、口述想法

##### 待办记录

创建待办事项，支持勾选完成。

**创建方式**：
- 点击工具栏的待办按钮
- 输入待办内容
- 点击复选框标记完成

**适用场景**：任务清单、提醒事项、行动计划

#### 1.3 标签管理

通过标签对记录进行分类和组织：

- **创建标签**：点击标签管理按钮，新建标签并命名
- **拖拽排序**：按住标签可拖拽调整顺序
- **折叠展开**：点击标签可折叠或展开记录列表
- **查看数量**：标签旁显示该分类下的记录数量

#### 1.4 记录操作

**单条记录操作**（右键菜单）：
- 查看详情
- 编辑内容
- 删除记录
- 拖拽到编辑器中引用

**批量操作**：
- 点击多选按钮进入批量选择模式
- 批量删除
- 批量转移标签
- 批量导出

### 二、整理功能

#### 2.1 整理功能入口

- 使用工具栏中的整理功能，可以使用整理模板。
- 使用 AI Agent 模式，通知 AI 使用哪些记录进行整理。

### 2.2 整理流程

#### 步骤一：选择整理范围

可选择以下时间范围的记录进行整理：

| 范围 | 说明 |
|------|------|
| 今天 | 整理今天创建的记录 |
| 本周 | 整理本周创建的记录 |
| 本月 | 整理本月创建的记录 |
| 近三月 | 整理最近三个月的记录 |
| 今年 | 整理今年创建的记录 |
| 全部 | 整理所有记录 |

#### 步骤二：选择整理模板

支持多个整理模板，每个模板包含不同的整理策略和提示词。在设置中可以自定义模板：
- 模板标题
- 整理时间范围
- AI 提示词内容

#### 步骤三：AI 智能整理

AI 会自动分析选定范围内的记录，按照逻辑关系和时间顺序进行组织，生成结构化的 Markdown 文档。

**整理特点**：
- 自动识别不同类型的记录
- 图片记录插入到合适位置
- 链接记录整理为参考章节
- 扫描记录使用 OCR 文本内容
- 保持内容的连贯性和可读性

#### 步骤四：保存文档

整理完成后，系统会自动创建新的 Markdown 文件并保存。文档标题根据模板设置自动生成。

### 2.3 自定义整理模板

在设置 > 工具栏设置 > 记录整理模板中，可以创建和管理自定义模板：

1. **模板标题**：为模板命名
2. **时间范围**：设置默认整理的时间范围
3. **提示词内容**：自定义 AI 整理的指令和要求

通过自定义模板，可以让 AI 按照你的偏好和需求来整理记录。

### 三、高级技巧

#### 3.1 快捷键

使用全局快捷键 Command or Alt + 对应工具排序（1～N），快速调起记录功能，无需切换窗口。

#### 3.2 拖拽引用

将记录直接拖拽到编辑器中，快速引用到文章中。

#### 3.3 搜索与筛选

- 按标签筛选记录
- 按记录类型筛选
- 按时间范围筛选

#### 3.4 数据同步

支持将记录数据同步到以下平台：
- GitHub
- Gitee
- GitLab
- Gitea

确保跨设备数据一致性和安全性。

#### 3.5 回收站

删除的记录会进入回收站，可以在一定时间内恢复或永久删除。

### 五、常见问题

**Q: 记录是否支持导出？**

A: 支持批量导出记录，可在多选模式下选择导出功能。

**Q: 整理后的文档可以编辑吗？**

A: 可以，整理完成后生成的是标准 Markdown 文件，可以自由编辑和修改。

**Q: 录音记录的语言支持哪些？**

A: 支持多种语言的语音转文字，具体取决于配置的 AI 模型。

**Q: 如何删除不再需要的标签？**

A: 在标签管理中可以删除标签，删除前可选择是否保留该标签下的记录。

**Q: 记录数据存储在哪里？**

A: 记录数据存储在本地数据库中，同时支持通过 Git 同步到远程仓库进行备份。
