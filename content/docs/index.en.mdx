### About NoteGen

NoteGen is a cross-platform Markdown AI Agent note-taking software, dedicated to bridging the gap between recording and writing with LLM.

It's completely open source and free, supporting Windows, macOS, Linux, Android, iOS and other mainstream platforms, with an installation package of only about 20MB. All data is stored locally, no login required, ensuring privacy and security.

### Core Features

#### Recording Tools

Capture fragmented thoughts anytime, never miss an idea:

- **Text Recording** - Quickly record text content
- **Screenshot Recording** - Use VLM or OCR technology to quickly capture screen content
- **Image Recording** - Save images with image recognition, insert to appropriate position when organizing notes
- **Link Recording** - Save web links and automatically extract page information
- **File Recording** - Support text recognition for plain text, PDF, etc.
- **Voice Recording** - Voice-to-text functionality automatically converts spoken content to text

#### AI Organization

Transform fragments into complete notes:

- **One-Click Generation** - Integrate all related records into complete notes
- **Drag to Insert** - Drag records into existing notes with ease
- **Tag Management** - Organize and manage records with tags for easy access
- **Organization Templates** - Flexible configuration with time range and custom prompts

#### AI Agent

Make AI your writing assistant, intelligently assisting with content creation and document management:

- **Intelligent Note Editing** - Agent can accurately read document contents, understand writing context, and make intelligent edits
- **MCP** - Through Model Context Protocol, Agent can access external tools, databases and services
- **Skills** - Built-in rich skill system, supports customizable workflows and task chains
- **RAG** - Build local knowledge base from your notes, using vector and re-ranking models for intelligent retrieval

#### Other Features

- **Markdown** - Supports standard Markdown and extends syntax, such as math formula, mind map, chart, flow chart, etc.
- **WYSIWYG** - Powered by Vditor editor, supports WYSIWYG mode with instant rendering experience like Typora
- **Themes** - Supports 12 preset themes with customizable theme colors, includes dark mode

### Quick Start

1. **Download & Install** - Visit [Download Page](/download) to get the installation package for your platform
2. **Start Recording** - Built-in free AI models, no configuration required
3. **AI Organization** - Let AI help you organize fragmented records into complete notes
4. **(Optional) Configure Model** - Set up your preferred AI model in settings
