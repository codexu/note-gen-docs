NoteGen is an open-source, cross-platform Markdown AI notes app that connects quick capture with long-form writing. Notes, records, chats, and settings are stored on your device. When you use an online model, sync provider, image host, or MCP server, the relevant data is sent to that service. See [Privacy and data](/en/docs/privacy).

## Supported platforms

| Platform | Distribution | Notes |
| --- | --- | --- |
| Windows | Installer | x64 desktop app |
| macOS | DMG | Apple Silicon and Intel |
| Linux | AppImage, Deb, RPM | x86_64 desktop app |
| Android | APK | ARM64 package currently provided |
| iOS | TestFlight | Availability may change; check the download page |

Visit the [download page](/en/download) for the current release. Some capabilities differ by platform; for example, command-based MCP servers are desktop-only.

## Core workflow

### 1. Capture

Collect text, screenshots, images, links, files, recordings, and todos. Organize records with tags, search, time/type filters, and AI organization templates.

### 2. Write

Use the Tiptap-based Markdown editor with tables, task lists, code blocks, math, Mermaid, images, search and replace, outlines, and multi-format export. AI tools can complete, polish, shorten, expand, and translate text.

### 3. Chat and Agent

Chat handles questions and content generation. Agent can read or modify notes, folders, tags, records, and memories after you approve the relevant tools. MCP, Skills, the knowledge base, and long-term memory extend its context and capabilities.

## Quick start

1. Install NoteGen from the [download page](/en/download).
2. Open Record and create your first text or voice record.
3. Open Writing and create a Markdown file, or organize records into a note.
4. Try the built-in free models, or add your own OpenAI-compatible provider under Settings → Model Configuration.
5. For multiple devices, configure GitHub, Gitee, GitLab, Gitea, S3, or WebDAV under Settings → Sync.
