### 添加模板

点击「添加模板」按钮，创建新的整理模板。

#### 配置项

| 配置项 | 说明 |
|-------|------|
| 名称 | 模板显示名称 |
| 范围 | 模板应用的记录范围 |
| 状态 | 开启或关闭模板 |
| 内容 | Prompt 模板内容 |

#### 范围选项

选择模板应用的时间范围：
- 全部
- 今天
- 近一周
- 近一月
- 近三个月
- 近一年

### 编辑模板

点击编辑按钮修改模板配置。

### 删除模板

点击删除按钮移除模板（默认模板无法删除）。

### 使用场景

整理模板可以用于：
- 自动生成周报/月报
- 按时间段整理笔记
- 格式化输出特定结构的内容
- 自动化文档整理流程
