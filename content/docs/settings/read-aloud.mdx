---
title: 朗读

朗读可用于 Chat 回复和编辑器选中文本。它与录音转写分开配置。

## 运行方式

| 模式 | 行为 |
| --- | --- |
| 自动（推荐） | 优先使用系统语音，不可用时尝试已配置的模型 |
| 仅本地 | 只使用操作系统提供的语音，不发送文本到模型服务 |
| 仅模型 | 使用配置的 TTS 模型，需要网络和对应服务凭证 |

## 朗读模型与语速

模型模式需要在模型配置中添加 `tts` 类型模型，再选择为朗读模型。语速范围为 0.5x 至 2.0x，默认 1.0x。

如果本地语音没有所需语言，请先在操作系统中安装对应语音，或切换到支持该语言的 TTS 模型。
