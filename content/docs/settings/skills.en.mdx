### What are Skills

Skills are reusable AI功能 modules that implement specific tasks through preset Prompt templates. AI automatically matches and calls appropriate Skills based on conversation content.

### Global Skills

Global Skills are available in all workspaces and suitable for common task scenarios.

### Managing Skills

#### Import Skill

Click the "Import Skill" button and select a ZIP format Skill package to import.

#### Skill Card

Each Skill displays the following information:
- Skill name and description
- Trigger conditions
- Enable/Disable status

#### Actions

| Action | Description |
|-----|------|
| Enable/Disable | Control whether the Skill is available |
| Edit | Modify Skill configuration |
| Delete | Remove the Skill |

### Use Cases

Skills can be used for:
- Automated document processing
- Batch format conversion
- Content analysis and summarization
- Custom workflows
- Scheduled task execution

### Notes

- Imported Skill packages must comply with the specification format
- Disabled Skills will not be called by AI
- Please confirm before deleting a Skill
