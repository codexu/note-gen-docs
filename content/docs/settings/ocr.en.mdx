### Configuration

Language packages, select the language packages you need. Chinese and English are supported by default.

Currently uses [Tesseract](https://github.com/tesseract-ocr/tesseract) as the OCR engine, [query languages](https://tesseract-ocr.github.io/tessdoc/Data-Files#data-files-for-version-400-november-29-2016).

Separated by commas, for example: `chi_sim,eng`. Adding new language packages will automatically download the corresponding language packages.
