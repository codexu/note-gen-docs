### Configuration

| Model | Description |
|-------|-------------|
| Primary Model | Used for AI chat, will be used when other models are not configured |
| Quick Complete | Used for text intelligent completion feature |
| Record Description | Used to generate record descriptions, such as records after OCR recognition |
| Commit | Used to automatically generate Git commit messages |

### Usage Recommendations

Some scenarios don't require powerful models; you can use low-cost models for faster response and reduced costs.

- **Primary Model**: Recommend using a model with strong comprehensive capabilities
- **Quick Complete**: Recommend using a model with fast response speed
- **Record Description**: Can use lightweight models
- **Commit**: Can use specialized code generation models or general-purpose models
