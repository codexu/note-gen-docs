### 什么是 Skills

Skills 是由 Anthropic 推出的**开放标准**，它是包含指令、脚本和资源的文件夹，AI 代理可以发现并使用这些内容来更准确、高效地完成任务。

虽然 AI 代理的能力日益增强，但往往缺乏可靠执行实际工作所需的上下文。Skills 通过为代理提供程序性知识和按需加载的公司、团队和用户特定上下文来解决这个问题。

访问一组 Skills 的代理可以根据其正在处理的任务扩展自身能力。

### 全局 Skills

全局 Skills 在所有工作区中可用，适用于通用的任务场景。

### 本地工作区 Skills

除了全局 Skills 外，你也可以在本地工作区中创建专属的 Skills：

1. 在工作区根目录下创建 `skills` 文件夹
2. 在 `skills` 文件夹内按照 Agent Skills 规范创建子文件夹，每个子文件夹代表一个 Skill
3. 每个 Skill 包含必要的指令、脚本和资源文件

本地 Skills 的结构与全局 Skills 完全一致，但仅对当前工作区可用。

### 管理 Skills

#### 导入 Skill

点击「导入 Skill」按钮，选择 ZIP 格式的 Skill 包进行导入。

#### Skill 卡片

每个 Skill 显示以下信息：
- Skill 名称和描述
- 触发条件
- 启用/禁用状态

#### 操作

| 操作 | 说明 |
|-----|------|
| 启用/禁用 | 控制 Skill 是否可用 |
| 编辑 | 修改 Skill 配置 |
| 删除 | 移除 Skill |

### 使用场景

Skills 可以用于：
- 自动化文档处理
- 批量格式转换
- 内容分析和总结
- 自定义工作流程
- 定期任务执行

### 开放标准

Agent Skills 格式最初由 Anthropic 开发，作为开放标准发布，已被越来越多的代理产品采用。该标准对生态系统的贡献开放。

### 注意事项

- 导入的 Skill 包需要符合规范格式
- 禁用的 Skill 不会被 AI 调用
- 删除 Skill 前请确认不再需要
