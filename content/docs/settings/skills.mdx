### 什么是 Skills

Skills 是可复用的 AI 功能模块，通过预设的 Prompt 模板实现特定任务。AI 会根据对话内容自动匹配并调用合适的 Skill。

### 全局 Skills

全局 Skills 在所有工作区中可用，适用于通用的任务场景。

### 管理 Skills

#### 导入 Skill

点击「导入 Skill」按钮，选择 ZIP 格式的 Skill 包进行导入。

#### Skill 卡片

每个 Skill 显示以下信息：
- Skill 名称和描述
- 触发条件
- 启用/禁用状态

#### 操作

| 操作 | 说明 |
|-----|------|
| 启用/禁用 | 控制 Skill 是否可用 |
| 编辑 | 修改 Skill 配置 |
| 删除 | 移除 Skill |

### 使用场景

Skills 可以用于：
- 自动化文档处理
- 批量格式转换
- 内容分析和总结
- 自定义工作流程
- 定期任务执行

### 注意事项

- 导入的 Skill 包需要符合规范格式
- 禁用的 Skill 不会被 AI 调用
- 删除 Skill 前请确认不再需要
