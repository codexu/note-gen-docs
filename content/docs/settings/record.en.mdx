## Record description model

This model processes OCR and related recognition results into descriptions that are easier to browse and organize. A small, fast chat model is usually sufficient. NoteGen falls back to the primary model when none is selected.

## Record toolbar

The toolbar can include text, voice, scan, image, link, file, and todo controls. Open Settings to:

- show or hide controls;
- drag them into a different order;
- configure desktop and mobile layouts separately.

Desktop tray shortcuts also follow the record-toolbar order. Hiding a control does not delete existing records.
