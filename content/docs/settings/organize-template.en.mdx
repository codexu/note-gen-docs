### Add Template

Click the "Add Template" button to create a new organize template.

#### Configuration Items

| Item | Description |
|------|-------------|
| Name | Template display name |
| Scope | Record scope for template application |
| Status | Enable or disable template |
| Content | Prompt template content |

#### Scope Options

Select the time range for template application:
- All
- Today
- Past Week
- Past Month
- Past Three Months
- Past Year

### Edit Template

Click the edit button to modify template configuration.

### Delete Template

Click the delete button to remove a template (default templates cannot be deleted).

### Use Cases

Organize templates can be used for:
- Automatically generating weekly/monthly reports
- Organizing notes by time period
- Formatting output of specific structure content
- Automating document organization workflows
