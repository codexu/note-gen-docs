## 默认模型

选择日常 Chat 和 Agent 使用的主要聊天模型。未单独指定的对话场景会回退到主要模型。

## 对话工具栏

点击“设置”调整桌面端和移动端工具栏按钮的显示、顺序和可见性。可配置模型、Prompt、回复语言、关联标签、关联文件、MCP、知识库、剪贴板监听、新对话、清除上下文和清空对话。

“清除上下文”保留界面中的历史消息，但后续请求不再携带此前内容；“清空对话”会删除当前会话历史。

## 对话摘要

长对话可自动压缩为摘要以减少 Token 消耗。可以配置：

- 是否启用自动摘要；
- 摘要模型，留空时使用主要模型；
- 触发消息数与单条消息最小 Token；
- 不压缩的最近消息数量；
- 摘要长度和自定义 Prompt，使用 `{content}` 代表原内容。

摘要会改变后续请求携带的上下文细节。对精确引用要求较高时，可以关闭自动摘要或开始新会话。
