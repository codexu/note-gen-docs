### Description

- Proxy, used to resolve network issues. Recommended to restart the application after configuration.
- Clear data information, including system configuration information, database (including records).
- Clear files (default workspace), including images, articles.
- Configuration file (JSON format) import and export. Importing a configuration file will overwrite the current configuration and takes effect after restart.
