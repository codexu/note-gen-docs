## Typewriter Mode

In typewriter mode, the editor simulates typewriter effects to help you better immerse yourself in writing.

## Default Enable Outline

When enabled, the editor will display the outline by default.

## Outline Position

Set the outline position, with options for left and right sides.

## Line Number Display

When enabled, the editor will display line numbers on the left side, making it easier to locate and reference code blocks or long documents.

## Page View

Set the editor's page view mode with options:
- **Edit Mode**: Clean view focused on writing
- **Preview Mode**: Real-time preview of rendered effects
- **Split Mode**: Side-by-side display of editing and preview
