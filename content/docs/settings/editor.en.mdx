### Typewriter Mode

In typewriter mode, the editor simulates a typewriter effect to help you better immerse yourself in writing.

### Default Enable Outline

When enabled, the editor will display the outline by default.

### Outline Position

Set the outline position, available on the left or right side.

### Line Number Display

When enabled, the editor will display line numbers on the left side, facilitating positioning and referencing code blocks or long documents.

### Page View

Set the editor's page view mode, options include:
- **Edit Mode**: A clean view focused on writing
- **Preview Mode**: Real-time preview of rendered effects
- **Split Mode**: Split-screen display showing both editing and preview
