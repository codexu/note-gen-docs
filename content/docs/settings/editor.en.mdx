## AI completion model

Select the chat model used for inline editor suggestions. When a suggestion appears, use the configured accept shortcut to insert it or the cancel shortcut to dismiss it. Fast models with stable output usually feel best.

## Centered content

When enabled, the document remains centered with margins on both sides. Disabling it uses more of the available editor width. This changes layout only, not Markdown content.

## Outline

- **Enable outline by default** shows the heading outline when a file opens.
- **Outline position** places it on the left or right.

The outline is generated from document headings. Select a heading to jump to it.

## Undo and redo buttons

Choose whether undo and redo appear in the editor tab bar. Keyboard shortcuts continue to work when the buttons are hidden.
