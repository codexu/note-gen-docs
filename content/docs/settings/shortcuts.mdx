### 可配置快捷键

| 功能 | 说明 |
|-----|------|
| 截图记录 | 快速启动屏幕截图记录 |
| 链接记录 | 快速启动链接剪贴板记录 |
| 文本记录 | 快速启动文本剪贴板记录 |
| 窗口置顶 | 切换窗口置顶状态 |

### 配置方式

1. 点击快捷键输入框
2. 按下想要设置的快捷键组合
3. 系统自动保存设置

### 快捷键建议

推荐使用不常用且易于按下的组合：
- `Ctrl/Cmd + Shift + 字母`
- `Ctrl/Cmd + Alt + 字母`
- `F1-F12` 功能键

### 注意事项

- 避免与系统快捷键冲突
- 某些组合可能已被其他应用占用
- 如果快捷键不生效，请检查是否有冲突
