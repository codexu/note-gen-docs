## Global shortcuts

Global shortcuts are registered with the OS and can trigger while NoteGen is in the background. They can show the window, open quick text capture, and launch record tools in toolbar order.

Modifier keys differ across systems. If registration fails, check for an OS or application conflict.

## Editor shortcuts

Editor shortcuts work only while the Markdown editor is focused and are grouped into:

- basic editing: undo, redo, find/replace;
- formatting: paragraph, headings 1–6, bold, italic, strike, code, quote, and more;
- AI: accept/dismiss completion and open AI or translation actions;
- table commands for adding/removing rows and columns;
- insertion commands for links, images, math, and other content.

The interface shows suggested combinations and detects conflicts between editor and global shortcuts.

## Change and reset

Focus a shortcut field and press the new combination. Clear one entry or restore defaults. Avoid plain letters, numbers, and navigation keys because they interfere with typing.
