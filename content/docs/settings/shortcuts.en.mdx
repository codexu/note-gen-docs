### Configurable Shortcuts

| Function | Description |
|----------|-------------|
| Screenshot Recording | Quickly launch screenshot recording |
| Link Recording | Quickly launch link clipboard recording |
| Text Recording | Quickly launch text clipboard recording |
| Always on Top | Toggle window always-on-top state |

### Configuration Method

1. Click the shortcut input field
2. Press the desired key combination
3. System automatically saves the setting

### Shortcut Recommendations

Recommended to use uncommon and easy-to-press combinations:
- `Ctrl/Cmd + Shift + Letter`
- `Ctrl/Cmd + Alt + Letter`
- `F1-F12` Function keys

### Notes

- Avoid conflicts with system shortcuts
- Some combinations may already be occupied by other applications
- If a shortcut doesn't work, please check for conflicts
