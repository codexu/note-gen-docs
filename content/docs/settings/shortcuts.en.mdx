### Configurable Shortcuts

| Function | Description |
|-----|------|
| Screenshot Record | Quick launch screenshot recording |
| Link Record | Quick launch link clipboard recording |
| Text Record | Quick launch text clipboard recording |
| Always on Top | Toggle window always-on-top state |

### How to Configure

1. Click the shortcut input box
2. Press the desired key combination
3. System automatically saves the settings

### Shortcut Suggestions

Recommended to use uncommon and easy-to-press combinations:
- `Ctrl/Cmd + Shift + Letter`
- `Ctrl/Cmd + Alt + Letter`
- `F1-F12` function keys

### Notes

- Avoid conflicts with system shortcuts
- Some combinations may already be occupied by other applications
- If shortcuts don't work, please check for conflicts
