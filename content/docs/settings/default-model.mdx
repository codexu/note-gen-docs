## 模型回退

主要聊天模型是对话类功能的默认回退。场景未选择模型时，通常使用主要模型；明确选择“不使用模型”时则关闭对应 AI 能力。

## 配置位置

| 场景 | 设置位置 | 建议 |
| --- | --- | --- |
| Chat / Agent | 对话设置 | 使用综合能力和工具调用较好的聊天模型 |
| 长对话摘要 | 对话设置 | 使用成本较低、擅长总结的模型 |
| 编辑器补全 | 编辑器设置 | 使用低延迟聊天模型 |
| 记录描述 | 记录设置或默认模型 | 使用轻量聊天模型 |
| 知识库嵌入/重排 | 知识库 | 必须匹配 embedding / rerank 类型 |
| 图片理解 | 图像识别 | 选择支持视觉输入的模型 |
| 语音转文字 | 音频设置 | 选择 stt 类型模型 |
| 文本朗读 | 朗读设置 | 选择 tts 类型模型，或使用系统语音 |

更换嵌入模型后，建议重新计算知识库向量，避免不同模型生成的向量混用。
