title: 默认模型

---

### 配置

| 模型 | 说明 |
|-----|------|
| 主要模型 | 用于 AI 对话，其他模型未配置时将使用主要模型 |
| 快速补全 | 用于文本智能补全功能 |
| 记录描述 | 用于生成记录的描述信息，如 OCR 识别后的记录 |
| Commit | 用于自动生成 Git 提交信息 |

### 使用建议

某些场景无需强大的模型，可以使用低成本模型，实现快速响应且降低使用成本。

- **主要模型**：建议使用综合能力较强的模型
- **快速补全**：建议使用响应速度快的模型
- **记录描述**：可以使用轻量级模型
- **Commit**：可以使用专门的代码生成模型或通用模型
