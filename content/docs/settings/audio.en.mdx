## Speech Recognition (STT)

Speech recognition converts your voice into text, enabling voice-based recording and input.

### Configuration

- **STT Model**: Select the AI model for speech recognition. Requires pre-configured speech-type models in model settings
- **Language**: Choose the recognition language type, supporting Chinese, English, and other languages

### Use Cases

- Quickly record thoughts via voice on the record page
- Replace keyboard input for improved efficiency
- Ideal for mobile usage

## Text-to-Speech (TTS)

Text-to-speech converts text into spoken audio with adjustable playback speed.

### Configuration

- **TTS Model**: Select the AI model for speech synthesis. Requires pre-configured speech-type models in model settings
- **Speed**: Adjust playback speed, ranging from 0.5x (slow) to 2.0x (fast)
  - 0.5x - Suitable for learning and comprehension
  - 1.0x - Normal speed (default)
  - 1.5x - Fast browsing
  - 2.0x - Speed reading

### Use Cases

- Read aloud note content for review
- Audio playback of long documents
- Free your eyes and consume information through listening
- Suitable for commuting, exercising, and other scenarios

## Important Notes

1. **Model Configuration**: Before using audio features, ensure speech-type models have been added in model configuration
2. **Network Requirement**: Speech recognition and synthesis require network connection, please ensure stable connectivity
3. **Quality Optimization**: Choosing appropriate models yields better recognition accuracy and voice naturalness
4. **Speed Recommendation**: First-time users should start with normal speed and adjust as needed

## Quick Actions

- In the writing page, select text to use the speech playback feature
- On the record page, click the voice button to start voice input directly
