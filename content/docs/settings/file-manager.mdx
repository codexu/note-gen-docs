## 工作区

工作区是 NoteGen 读取和保存 Markdown、目录及本地资源的根目录。可选择任意有读写权限的文件夹，也可以恢复默认路径。

默认工作区位于操作系统为 NoteGen 分配的应用数据目录中，具体路径会随系统、安装方式和运行环境变化，建议以设置页显示的“当前工作区”为准。

## 切换与历史

- 点击“选择工作区目录”浏览新目录。
- 从历史路径快速切换，或删除单条、清空全部历史。
- 选择“重置为默认路径”返回应用工作区。
- 更改后重启应用，使文件树、Skills、索引和编辑器完全加载新路径。

切换不会移动旧工作区文件，也不会自动合并两个工作区。

## 写作资源路径

资源路径设置的是工作区内用于保存编辑器本地图片等资源的目录名称，例如 `assets` 或 `images`。建议使用简单的相对目录名，以便整个工作区移动、备份和同步。

更改名称只影响之后保存的资源，不会批量移动已有文件或修改旧 Markdown 中的链接。
