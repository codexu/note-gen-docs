## 什么是 Prompt？

Prompt（提示词）是您与 AI 助手交互时的系统指令，它定义了 AI 在对话中的角色、行为方式和响应风格。

通过预设的 Prompt 模板，您可以：
- **节省时间**：无需每次重复输入相同的指令
- **保持一致**：确保同类任务得到风格统一的响应
- **提升质量**：精心设计的 Prompt 能获得更精准的回答

常见的应用场景包括：代码审查、文案写作、语言翻译、数据分析、学习辅导等。

## 使用 Prompt

在 Chat 对话界面中，您可以通过下拉菜单快速切换使用不同的 Prompt 模板。

### 切换 Prompt

点击对话框顶部的 Prompt 选择器，从列表中选择您需要的模板。切换后，新的 Prompt 会立即应用到当前对话，影响后续的所有响应。

### 使用技巧

- **选择合适的 Prompt**：根据当前任务类型选择最匹配的模板
- **组合使用**：可以在对话中随时切换 Prompt，适应不同的交流需求
- **参考默认模板**：系统内置的 Prompt 是经过优化的，可以作为编写自定义 Prompt 的参考

## 配置 Prompt

### 添加 Prompt

点击「添加 Prompt」按钮，输入标题和内容，创建自定义提示词模板。

**标题建议**：
- 简洁明了，能快速识别用途（如「代码审查专家」「英文写作助手」）
- 避免过长，建议 2-8 个字符

**内容建议**：
- 明确角色定位：「你是一位专业的...」
- 指定输出格式：「请以 JSON 格式返回...」「请用表格列出...」
- 设置约束条件：「回答要简洁」「避免使用技术术语」

### Prompt 优化

编辑时可以使用 AI 优化功能，自动改进 Prompt 内容的清晰度和有效性。

AI 优化会：
- 补充角色设定和上下文
- 优化指令的表达方式
- 添加输出格式要求
- 增强约束条件

优化后请检查是否符合您的预期，必要时可手动调整。

### 编辑 Prompt

点击编辑按钮可以修改现有 Prompt 的标题和内容。

修改 Prompt 后：
- 已使用该 Prompt 的对话不受影响
- 新对话将使用更新后的内容
- 建议先复制备份再大幅修改

### 删除 Prompt

点击删除按钮移除不需要的 Prompt（默认 Prompt 无法删除）。

> **注意**：删除操作不可恢复，请谨慎操作。建议将常用的 Prompt 保留，不常用的可以先禁用而非删除。
