### 添加 Prompt

点击「添加 Prompt」按钮，输入标题和内容，创建自定义提示词模板。

#### Prompt 优化

编辑时可以使用 AI 优化功能，自动改进 Prompt 内容的清晰度和有效性。

### 编辑 Prompt

点击编辑按钮可以修改现有 Prompt 的标题和内容。

### 删除 Prompt

点击删除按钮移除不需要的 Prompt（默认 Prompt 无法删除）。

### Awesome Prompts

点击「Awesome Prompts」链接可以获取优秀的 Prompt 模板参考：https://github.com/f/awesome-chatgpt-prompts
