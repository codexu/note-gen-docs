### Add Prompt

Click the "Add Prompt" button, enter title and content to create custom prompt templates.

#### Prompt Optimization

When editing, you can use AI optimization feature to automatically improve the clarity and effectiveness of prompt content.

### Edit Prompt

Click the edit button to modify existing prompt title and content.

### Delete Prompt

Click the delete button to remove unwanted prompts (default prompts cannot be deleted).

### Awesome Prompts

Click the "Awesome Prompts" link to get excellent prompt template references: https://github.com/f/awesome-chatgpt-prompts
