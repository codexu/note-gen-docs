## Workspace

The workspace is the root directory where NoteGen reads and writes Markdown, folders, and local assets. Select any directory with read/write permission or restore the default.

The default workspace is inside the OS application-data location assigned to NoteGen. Exact paths vary by OS, installation, and runtime, so use the Current Workspace value shown in settings.

## Switch and history

- Select a new directory with Choose Workspace.
- Switch through history, remove one entry, or clear all history.
- Reset to return to the application workspace.
- Restart so the file tree, Skills, indexes, and editor fully load the new path.

Switching does not move files or merge two workspaces.

## Writing asset path

This value is the workspace directory name used for local editor images and related assets, such as `assets` or `images`. Use a simple relative directory name so the workspace remains portable and easy to back up.

Changing it affects new assets only. Existing files are not moved and links in older Markdown are not rewritten.
