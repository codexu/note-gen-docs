### 本地备份

手动将应用数据导出为 ZIP 文件，或从备份文件恢复数据。

#### 导出备份

点击「导出备份」按钮，选择保存位置，应用会将所有数据打包为 ZIP 文件。文件名格式为 `note-gen-backup-YYYY-MM-DD_HH-mm-ss.zip`。

#### 导入备份

点击「导入备份」按钮，选择之前导出的 ZIP 文件，应用将恢复所有数据。导入后需要重启应用生效。

### WebDAV 备份

通过 WebDAV 协议将数据备份到云端服务器，实现跨设备数据同步。

#### 配置项

| 配置项 | 说明 |
|-------|------|
| 服务器地址 | WebDAV 服务器地址，例如：https://webdav.example.com |
| 用户名 | WebDAV 登录用户名 |
| 密码 | WebDAV 登录密码 |
| 备份路径 | 云端备份路径，例如：/note-gen/backup |

#### 连接状态

系统会自动检测 WebDAV 连接状态，显示为：
- **成功**：连接正常，可以备份和同步
- **检测中**：正在验证连接
- **失败**：连接失败，请检查配置

#### 备份操作

| 操作 | 说明 |
|-----|------|
| 备份至 WebDAV | 将本地数据上传到 WebDAV 服务器 |
| 从 WebDAV 同步 | 从 WebDAV 服务器下载数据到本地 |
| 创建目录 | 如果备份路径不存在，可以创建新目录 |

#### 使用场景

WebDAV 备份适用于：
- 使用自建云存储服务（如 Nextcloud、ownCloud）
- 企业内部的文件服务器
- 支持 WebDAV 的网盘服务

#### 注意事项

- WebDAV 备份不支持历史回滚功能
- 请确保服务器地址和凭证配置正确
- 建议使用 HTTPS 协议以保证数据安全
