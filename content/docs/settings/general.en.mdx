## Appearance

- Choose light, dark, or system theme.
- Use a preset palette or custom theme color.
- Select the font family used by application content.
- UI scale changes the whole app; content, file-manager, and record text sizes can be adjusted separately.

Scale and local fonts are device-specific and excluded from cross-device settings sync by default.

## Language

Changing the interface language updates most UI immediately. Third-party output, model responses, and document content are not translated automatically. Chat response language is configured separately in the chat toolbar.

## System tray

Desktop can keep NoteGen in the system tray. The tray menu can show the window and start quick record actions. Quick actions follow the enabled controls and order in Record settings. Mobile has no system tray.

## Toolbar settings

Chat and record toolbar configuration now lives on the Chat and Record settings pages, with separate desktop and mobile layouts.
