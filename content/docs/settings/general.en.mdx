### Interface Settings

#### Theme

Choose the application theme appearance:

| Option | Description |
|-----|------|
| Light | Use light theme |
| Dark | Use dark theme |
| Follow System | Automatically follow operating system theme |

#### Preset Themes

NoteGen provides 12 carefully designed preset themes, including light and dark themes, to meet different scenarios and preferences.

#### Custom Theme Color

Support custom theme accent colors for a personalized interface.

#### Language

Select application interface language. Supports multi-language switching.

#### Interface Zoom

Adjust the zoom ratio of the entire application interface:

| Option | Description |
|-----|------|
| 75% | Smaller display, suitable for large screens |
| 100% | Standard size (default) |
| 125% | Magnify 1.25x |
| 150% | Magnify 1.5x |

#### Content Zoom

Adjust the display size of the content in the editing area separately.

#### File Manager Text Size

Adjust the text display size in the left file tree.

#### Record Text Size

Adjust the text display size in the record list.

#### Enable Tray

When enabled, the application will appear in the system tray when minimized, supporting tray shortcuts.

### Tool Settings

#### Chat Toolbar Configuration

Configure the display content and order of the chat toolbar for PC and mobile respectively.

#### Record Toolbar Configuration

Configure the display content and order of the record page toolbar.
