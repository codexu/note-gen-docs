### Interface Settings

#### Theme

Select the application theme appearance:

| Option | Description |
|--------|-------------|
| Light | Use light theme |
| Dark | Use dark theme |
| Follow System | Automatically follow operating system theme switching |

#### Preset Themes

NoteGen provides 12 carefully designed preset themes, including both light and dark themes to meet different scenarios and preferences.

#### Custom Theme Color

Support customizing theme accent colors to create a personalized interface.

#### Language

Select the application interface language. Supports multi-language switching.

#### Interface Zoom

Adjust the zoom level of the entire application interface:

| Option | Description |
|--------|-------------|
| 75% | Smaller display, suitable for large screens |
| 100% | Standard size (default) |
| 125% | Enlarge 1.25x |
| 150% | Enlarge 1.5x |

#### Content Zoom

Adjust the display size of the editing area content independently.

#### File Manager Text Size

Adjust the text display size in the left file tree.

#### Record Text Size

Adjust the text display size in the record list.

#### Enable Tray

When enabled, the application will display in the system tray when minimized, supporting tray quick actions.

### Tool Settings

#### Chat Toolbar Configuration

Configure the display content and order of the chat toolbar for PC and mobile respectively.

#### Record Toolbar Configuration

Configure the display content and order of the toolbar on the record page.
