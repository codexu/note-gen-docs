NoteGen 是一款开源、跨平台的 Markdown AI 笔记应用，使用 AI 连接碎片记录与长文写作。笔记、记录、聊天和设置保存在本机；当你使用在线模型、同步、图床或 MCP 时，相关数据会发送到你选择的服务。详见[隐私与数据](/cn/docs/privacy)。

## 支持平台

| 平台 | 获取方式 | 说明 |
| --- | --- | --- |
| Windows | 安装程序 | x64 桌面端 |
| macOS | DMG | Apple Silicon 与 Intel |
| Linux | AppImage、Deb、RPM | x86_64 桌面端 |
| Android | APK | 当前提供 ARM64 安装包 |
| iOS | TestFlight | 可用性以下载页为准 |

访问[下载页面](/cn/download)获取当前版本。不同平台的可用功能可能不同，例如本地命令型 MCP 仅支持桌面端。

## 核心工作流

### 1. 记录

用文本、截图、图片、链接、文件、录音和待办快速收集素材。可以使用标签、搜索、时间和类型筛选管理记录，再通过整理模板生成 Markdown 笔记。

### 2. 写作

在基于 Tiptap 的 Markdown 编辑器中写作，支持表格、任务列表、代码块、数学公式、Mermaid、图片、搜索替换、文档大纲和多格式导出。AI 可以补全、润色、精简、扩写和翻译内容。

### 3. 对话与 Agent

Chat 用于问答和内容生成；Agent 可以在授权后读取和修改笔记、文件夹、标签、记录与记忆。MCP、Skills、知识库和长期记忆可以进一步扩展 Agent 的上下文与工具。

## 快速开始

1. 从[下载页面](/cn/download)安装 NoteGen。
2. 打开“记录”，创建第一条文本或录音记录。
3. 打开“写作”，新建 Markdown 文件，或使用“整理”生成笔记。
4. 直接使用内置免费模型体验基础能力，或在“设置 → 模型配置”添加自己的 OpenAI 兼容服务。
5. 如需跨设备使用，在“设置 → 同步配置”选择 GitHub、Gitee、GitLab、Gitea、S3 或 WebDAV。
